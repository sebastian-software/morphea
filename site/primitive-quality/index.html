Preview L1 / edge
Mean per-pixel color and edge-structure difference between the manifest-rendered preview and the source bitmap. 0 is identical; family budgets sit between 0.001 and 0.08.
SVG L1 / edge
The same two measures against the rasterized exported SVG, the file you would actually use.
BBox IoU
Bounding-box overlap between detected and expected geometry; 1.0 is a perfect match.
Anchors / nodes
