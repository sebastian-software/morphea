Local vectorization research

SVGs that feel intentionally constructed.

Curve is a semantic-first raster-to-SVG prototype. It favors editable primitives, clean strokes, cut-out semantics, and coherent scene structure over dense pixel-tracing output.

Read the plan View milestones

Pipeline

Geometry first, then learning and refinement.

1

Preprocess

Normalize alpha, reduce palette drift, bound analysis size, and keep real images tractable.

2

Anchor

Detect circles, strokes, rings, rectangles, quads, cut-outs, and regular grids before generic fitting.

3

Model

Build a scene graph with layers, groups, reservations, metrics, and export policy metadata.

4

Improve

Use reports, curated checks, pseudo-label review, retraining, and structure-preserving refinement.

Current baseline

Built around editability signals.

Primitive anchors

Circles, dots, rings, strokes, arcs, rectangles, rounded rectangles, quads, and perspective grids.

Cut-out semantics

White interior gaps can stay selectable as editable overlay strokes or SVG mask candidates.

Real-image runtime controls

Bounded max size, palette quantization, component deferral, timeout diagnostics, and profile reports.

Research loop

Synthetic data, classifier training, curated checks, review queues, retraining gates, and refinement.

Visual anchors

Simple shapes get priority because people notice when they are wrong.

A slightly lumpy circle or uneven grid makes an illustration feel broken immediately. Curve reserves simple forms early so later organic detail cannot fragment them.

Quickstart

Run the current baseline locally.

python -m pip install -e .
python -m unittest discover -s tests
curve vectorize input.png -o output.svg --color-tolerance 18
curve curated-check docs/real-images/suite.json -o runs/curated.json --run
